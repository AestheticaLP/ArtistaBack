## How to launch the project 🚀
### Clone the project 📋
In ssh
```sh
git clone git@github.com:AestheticaLP/ArtistaBack.git
```

### Go the project folder 📂
```bash
cd ArtistaBack/functions
```
### Install dependencies 🐡
```sh
npm install
```

### Run the project
```bash
npm run serve
```
